# GetHub — HubieQuest Demo

Turn goals into daily quests.
Build habits.
Grow your dream.

This is the public shipping demo for HubieQuest (Dream & Grow MVP).

## What It Does

- Generate daily quests based on a goal
- Create a Dream Quest (weekly milestone)
- Track XP and streaks
- Post progress to a social feed
- Collect early user feedback
